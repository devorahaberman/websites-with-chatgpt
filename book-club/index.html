Design, develop and disscuss together.

At UX Reads & Reacts, we believe that learning is a journey best taken together. This book club is a space where developers, designers, and curious minds come together to explore the world of coding, web development, and user experience through the lens of continuous learning. Each month, we dive into books, articles, and resources that spark conversation about design thinking, accessibility, emerging technologies, and the human side of building digital experiences. But we don’t just read — we react. Our discussions encourage members to share insights, challenge ideas, and connect theory with practice. Whether you’re a seasoned developer or just starting your coding journey, UX Reads & Reacts offers a welcoming community where knowledge grows, creativity thrives, and every voice matters.

Current Book

Build a Website with ChatGPT

By: Paul McFedries

Summary: This book is designed for anyone curious about creating websites without needing prior coding experience. It introduces readers to AI-assisted web development, showing how tools like ChatGPT can generate HTML, CSS, and JavaScript with simple prompts. You’ll learn how to:

  • Craft effective prompts to build functional web components
  • Convert text into images using DALL·E integration
  • Create navigation bars, galleries, and contact forms
  • Deploy fully functional sites online for free
  • Customize generated code to make unique, personalized websites

The book emphasizes accessibility and creativity, making it perfect for learners who want to combine design thinking with modern AI tools.

Page Count: ~250 pages
